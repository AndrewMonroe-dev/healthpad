Architecture notes: one canvas, one 2D context, layered draw passes per frame (mist → clipped sphere [plasma/cracks/shading] → shell → embers/sparks → pulse). A single deterministic crack/vein network doubles as glowing filaments (thriving) or damage cracks (failing) — same structure, recolored by the score, not swapped out. Health state (sliders) and visual state (what the renderer reads) are two separate objects — the slow tick converts one into the other; the fast tick never touches health state directly.
